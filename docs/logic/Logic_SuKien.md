# LOGIC NGHIỆP VỤ: SỰ KIỆN

## 1. LOẠI SỰ KIỆN
- **Theo thời gian:** Tết, Giáng sinh... (Chỉ diễn ra trong khoảng ngày X đến Y).
- **Theo khung giờ:** Giờ vàng (x2 EXP từ 20h-22h).

## 2. ĐIỀU KIỆN THAM GIA
- **Vé (Ticket):** Mỗi ngày cho 3 vé miễn phí. Mua thêm bằng Coin hoặc xem Ads.
- **Level:** Yêu cầu Level > X mới được tham gia.

## 3. PHẦN THƯỞNG
- **Currency:** Coin, Gem.
- **Items:** Khung Avatar (có thời hạn hoặc vĩnh viễn), Vé đổi tên.
- **Hiện vật:** Voucher (nếu tích hợp).

## 4. LOGIC TÍNH ĐIỂM SỰ KIỆN
- Bảng xếp hạng sự kiện tách biệt với BXH chính.
- Điểm sự kiện reset khi hết sự kiện.
- Quy tắc cộng dồn điểm event khác với RP (ví dụ chỉ cộng, không trừ khi thua).
